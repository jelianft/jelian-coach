🔑 API Key de Anthropic
¿Para qué sirve?
La API Key permite que el módulo IA Coach funcione de forma independiente, fuera de Claude.ai. Sin ella, la IA solo funciona dentro de claude.ai.

¿Dónde la obtengo?
Entra a console.anthropic.com → API Keys → Create Key
🔒 La key se guarda solo en tu navegador (localStorage). Nunca se envía a ningún servidor excepto directamente a Anthropic.
